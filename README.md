


# Batch Zoom Tool — 批量图像局部放大工具

[![Python Version](https://img.shields.io/badge/python-3.7+-blue.svg)](https://www.python.org/downloads/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

一个用于批量为图像添加局部放大面板的 Web 应用。适用于学术论文、医学影像、显微摄影等需要突出显示感兴趣区域的场景。

## 功能特性

- **智能放大镜** — 精确标记和放大图像中的感兴趣区域（ROI）
- **实时预览** — Canvas 实时渲染缩放框、放大镜和连接线，所见即所得
- **灵活样式** — 自定义连接线颜色、粗细、线型和连接角点
- **精准控制** — 支持相对坐标（0.0~1.0）进行跨分辨率兼容
- **四步工作流** — 上传 → 框选缩放区域 → 调整放大镜 → 配置连接线
- **批量处理** — 支持同时处理多张图像，一键打包下载
- **论文级质量** — 输出 300 DPI PNG，适合期刊发表

## 快速开始

### 环境要求

- Python 3.7+
- Flask
- matplotlib
- numpy

### 安装与启动

```bash
pip install flask matplotlib numpy
python app.py
```

浏览器访问 **http://127.0.0.1:8080**

> **macOS 用户注意：** 如果访问 `localhost:5000` 遇到 403 错误，这是 macOS AirPlay Receiver 占用了 5000 端口所致。本项目默认端口已设为 8080，请直接访问 `http://127.0.0.1:8080`。

### 四步工作流

#### 第一步：上传图像

点击上传区域或拖拽图像文件。支持 PNG、JPG、BMP、TIFF 格式，可一次上传多张。

#### 第二步：框选缩放区域

在图像上拖拽鼠标框选要放大的区域。画面叠加了 10% 间距的参考网格辅助定位。

- 直接拖拽框选区域
- 或通过下方输入框精确设置 `xmin, xmax, ymin, ymax` 值（范围 0.0~1.0）

#### 第三步：调整放大镜

Canvas 实时预览放大镜在图中的实际位置、大小和放大效果。

**九宫格快捷位置：**

| | 左 | 中 | 右 |
|---|---|---|---|
| **上** | `0.02, 0.55, 0.40, 0.40` | `0.29, 0.55, 0.40, 0.40` | `0.58, 0.55, 0.40, 0.40` |
| **中** | `0.02, 0.28, 0.40, 0.40` | `0.29, 0.28, 0.40, 0.40` | `0.58, 0.28, 0.40, 0.40` |
| **下** | `0.02, 0.02, 0.40, 0.40` | `0.29, 0.02, 0.40, 0.40` | `0.58, 0.02, 0.40, 0.40` |

**微调参数（滑块）：**

| 参数 | 说明 | 范围 |
|------|------|------|
| 左边距 | 放大镜左侧边缘位置 | 0.0~0.8 |
| 下边距 | 放大镜下边缘位置 | 0.0~0.8 |
| 宽度 | 放大镜宽度比例 | 0.1~0.7 |
| 高度 | 放大镜高度比例 | 0.1~0.7 |

#### 第四步：配置连接线

Canvas 实时预览完整效果：缩放框、放大镜、连接线同时显示在画面上。

- **黄色圆点 0~3** = 缩放框的四个角点
- **青色圆点 0~3** = 放大镜的四个角点
- **虚线** = 连接线，中点标注连接关系（如 `R0 -> I0`）

```
角点编号（视觉位置）：

     0 ──────── 1
     │          │
     │          │
     3 ──────── 2

 0 = 左上    1 = 右上
 3 = 左下    2 = 右下
```

**6 种连线预设方案：**

| 方案 | 配置 | 适用场景 |
|------|------|---------|
| 左上+左下 | `R0→I0, R3→I3` | 放大镜在左侧 |
| 右上+右下 | `R1→I1, R2→I2` | 放大镜在右侧 |
| 左上+右上 | `R0→I0, R1→I1` | 放大镜在上方 |
| 右下+左下 | `R2→I2, R3→I3` | 放大镜在下方 |
| 右侧→左侧 | `R1→I0, R2→I3` | 对角连接 |
| 左侧→右侧 | `R0→I1, R3→I2` | 对角连接 |

如果连接线出现交叉，切换另一种方案即可。

**线条样式：**

| 选项 | 说明 |
|------|------|
| 颜色 | 颜色选择器，默认红色 |
| 粗细 | 1~6，默认 2 |
| 线型 | 虚线 `--`（默认）、实线 `-`、点线 `:`、点划线 `-.` |

### 预览与导出

- **Canvas 实时预览** — 调整参数时画面同步更新，无需等待
- **后端精确预览** — 点击「生成精确预览」，调用 matplotlib 渲染与最终输出一致的高精度图片
- **批量处理** — 一次性处理所有上传的图像
- **打包下载** — 将批量处理结果打包为 ZIP 下载

## 参数说明

所有坐标参数使用相对比例（0.0~1.0），与图像分辨率无关，同一组参数适用于不同尺寸的图片。

### 缩放区域

```
ZOOM_RATIOS = (xmin, xmax, ymin, ymax)

xmin, xmax = 水平方向比例（0.0 = 左边缘, 1.0 = 右边缘）
ymin, ymax = 垂直方向比例（0.0 = 顶部, 1.0 = 底部）
```

### 放大镜位置与大小

```
INSET_RECT = [左边距, 下边距, 宽度, 高度]
```

## 支持格式

- **输入：** PNG、JPG、JPEG、BMP、TIFF、TIF
- **输出：** PNG，300 DPI，黑色背景

## 命令行模式

除 Web 界面外，也可直接使用 `batch_zoom.py` 命令行脚本。适合参数已确定、需要重复批量处理的场景。

```bash
# 修改 batch_zoom.py 顶部的 MODE 和参数后运行
python batch_zoom.py
```

通过 `MODE` 变量控制三步工作流：`"grid"` 生成网格参考图 → `"debug"` 生成角点调试图 → `"final"` 正式出图。
